## Description

Planet Amtcha is a simulation game where the player takes on the role of a human who crash-lands on an unfamiliar alien planet and opens a matcha shop to provide for the aliens. The core gameplay revolves around interpreting alien customer orders and inputting them into a POS system. However, communication is intentionally distorted, making it difficult to understand what each alien is asking for.

The main mechanic challenges players to decode unclear, fragmented, or altered language, simulating the experience of communicating with someone who has aphasia. Players must decide whether to guess an order, ask for clarification, or risk making mistakes, balancing speed, accuracy, and patience.

The overall player experience is meant to feel slightly confusing but engaging, encouraging empathy and awareness. By putting players in a position where communication is unreliable, the game highlights the challenges faced by individuals with aphasia while still maintaining a fun, interactive café-style gameplay loop.

## Setup and Interaction Instructions:

Setup:

The game is hosted online using GitHub Pages, so no installation is required. To run the game, simply open the provided GitHub Pages link in any web browser. Once the page loads, click the “Start” button to begin.

To play, players interact using their mouse. Alien customers will appear and present their orders. The player must interpret what the alien is trying to order, click “Okay!” and use the on-screen POS system to recreate it.

Players click buttons to select menu items (such as drink type, or add-ons) and then submit the order. If the order matches what the alien intended, their satisfaction level remains unchanged, if not, it counts as a mistake.

The gameplay focuses on quickly and accurately interpreting unclear communication, creating a balance between speed, patience, and decision-making as more customers arrive.

Interactions:

- Read the alien customer’s order carefully.
- Decipher and memorize the requested items or combinations.
- after pressing okay, you go into POS system and click the icons to match the order combinations, and submit before the time runs out
- To successfully complete the level/day, maintain the satisfaction meter above the required threshold
- Submit the order to serve the customer.
